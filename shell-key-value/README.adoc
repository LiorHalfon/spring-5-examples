This is a simple spring-boot 5 shell example using spring-boot-keyvalue (Map as database) and kotlin language.

.build and run
----
bash gradlew clean assemble
java -jar build/libs/*.jar

shell>help
shell>connect --name max
----

using:

. spring shell
. spring-data-keyvalue
. kotlin
